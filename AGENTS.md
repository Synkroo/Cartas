# Project Instructions

- No fuerces texto, imagenes, colores, tamanos ni referencias visuales por codigo salvo que se pida explicitamente. La UI debe poder editarse desde el Inspector.
- No generes GameObjects solo en ejecucion si son parte de la escena o del layout editable. Deben existir en escena o prefab para poder modificarlos.
- Excepcion permitida: instanciar prefabs dentro de un `Content` para representar datos de ScriptableObjects, como articulos de tienda, cartas de mazo o misiones.
